# DamageScope AI Support

For help with DamageScope AI, please open an issue in the GitHub repository:

https://github.com/lanray07/DamageScope-AI/issues

Include:

- Device model and iOS version
- App version
- A short description of the issue
- Steps to reproduce the issue
- Whether the issue relates to cases, photos, AI scan results, subscriptions, PDF export, or App Store setup

Do not include sensitive client information, private addresses, insurance details, or confidential photos in public support requests.
